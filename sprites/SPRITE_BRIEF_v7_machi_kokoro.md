## 🎮 この素材で作るシステム（先に理解してから描くこと）

**まちのげんきど（0〜100）**：プレイヤーがちゃんとお世話・お散歩するほど上がり、
実時間で放置するほど下がる。街全体がその鏡になる。

| げんきど | 街のようす |
|---|---|
| 0〜25 どんより | 花はしおれ、草は枯れ、水たまりが澱む。噴水は止まり、屋台は布をかぶる。住民は外に出ず、口調も冷たい |
| 25〜60 ふつう | いまの状態 |
| 60〜85 いきいき | つぼみが咲き、蝶が増え、住民がよく外にいる。口調があたたかい |
| 85〜100 きらきら | 満開・ガーランドと風船の飾り・虹の噴水・**旅のお客さまが遊びに来る** |

つまり同じ「花」「木」「噴水」の**状態ちがい差分**が必要。
差し替え用は**必ず元画像を参照に添付**して、同じ構図・同じサイズ感で描くこと。

## 手順
1. A → B → C → D → E → F の順。**B/Cは必ず元画像を参照添付**（同じ構図の状態差分にする）
2. 各カテゴリ1個目を基準に統一。白背景・1画像1個
3. シート（E）はキャラ規格チェック（3×3・右向き・足元・かわいさ基準）
4. カテゴリごとに完了報告

## 📝 実装メモ（Claude側・納品後）
- げんきど計算：お世話/会話/クエスト/レース/掃除で加点、実時間放置で減点（お世話状態と連動）
- 低いと：花→しおれ差分、草に枯れパッチ、噴水停止、屋台休業、住民は外に出ず**セリフが冷たい**
  （「…ひさしぶりだね」「${dName()}を だいじに してあげてよ」）
- 高いと：つぼみ→満開、装飾出現、虹の噴水、訪問者ポッポさん＆ビーちゃん来訪、
  住民のセリフが**あたたかく**（「${pName()}が いると まちが あかるいね！」）
- セリフの温度差は なかよし度（♥）とげんきどの両方を参照
