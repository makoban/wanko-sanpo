## 🎀 かわいさのルール（最重要・全キャラ厳守）

どうぶつの森らしい かわいさは、次の要素で決まる。1つずつ検品すること。

### 顔
1. **目は顔の下半分に置く**（頭のてっぺん寄りはNG。目が下にあるほど幼く見える）
2. **目と目の間を広くあける**（顔幅の1/3以上あける）
3. 目は大きな縦長の丸。真っ黒ベタ＋**右上に白いキャッチライト1〜2ドット**（これが命）
4. 口は目のすぐ下に**小さく**。ねこ口「ω」か、ちいさな「▽」の笑い口
5. **ほっぺに必ずピンクのチーク**（2×2ドット程度、左右対称）
6. 鼻は小さく（1〜3ドット）。犬・くま・ぶたは鼻を可愛さのチャームポイントにしてよい

### シルエット
7. **首はない**。頭がそのまま体に乗る
8. 全身どこにも尖った角を作らない。**すべて丸く**（耳先だけ例外可）
9. 頭：体 = 6：4（頭が全高の約60%）。頭は**縦より横に広い**まんじゅう型
10. 手足は短く丸く「もちもち」に。指は描かない
11. 立ちポーズは左右対称で、両手を体の横にちょこんと

### 色と質感
12. 彩度低めのパステル。基準色：クリーム#fff3e0 / ピンク#ff8fae / 緑#8ecf6f / 水色#aee3f5 / 黄#ffd166
13. 毛色はベタ塗り＋**おなか・マズル・耳の内側に明るい差し色**で立体感（グラデ禁止）
14. アウトラインは真っ黒でなく**こげ茶 #4a3228** 1px
15. 1キャラ最大12色

### チャームポイント（キャラごとに1つ）
16. 各キャラに**小物を1つ**つける（下の表参照）。小物が「そのキャラらしさ」を作る

## 📐 フォーマット（v2と同じ・厳守）

- 1キャラ = 1ファイル、**144×144px**（3列×3行、セル48×48px）、背景完全透過
- 行1：正面（立ち / 歩き右足前 / 歩き左足前）
- 行2：背面（同上）
- 行3：**右向き**の横（立ち / 歩きA / 歩きB）※全キャラ必ず右向き。左はゲーム側で反転
- キャラ身長 約40px、**足元は全コマ y=44 のベースラインに揃える**
- 歩き2コマは必ず逆の足が前。よちよち歩き（waddle）に見えるとなお良い
- アンチエイリアス・グラデーション・ぼかし禁止。正確なドット格子
- 保存先：`sprites/48/` に同名で（girl.png / dog.png / mimii.png / moka.png /
  mike.png / kerota.png / penta.png / tonko.png / meeko.png / noron.png）

## 🛠 制作手順

1. **ミミィ（うさぎ）を最初に作り、スタイル基準にする**
   - 参考として前回の `sprites/mimii.png` を添付しつつ、「同じキャラの、より可愛いリデザイン」と指定
2. 2体目以降は**ミミィv3のシートを毎回参照画像に添付**（「same art style, same proportions, same cell layout」）
3. 1体ごとに下の「かわいさチェックリスト」で検品。NGなら作り直し
4. デザインの同一性：**服の色・種族・名前の印象は変えない**（別キャラにしない）

## 🚫 やりがちなNG

- 目が頭の上寄り・目が近い（大人っぽく＝可愛くなくなる）
- キャッチライトなしの黒目（ぬいぐるみ感が消える）
- 純黒アウトライン（硬い印象になる。#4a3228を使う）
- 細い手足・首がある（人間っぽくなる）
- 彩度の高い原色（パステルにする）
- 任天堂の実在キャラ（しずえ・たぬきち等）に似せること
