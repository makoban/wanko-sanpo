# ドット絵スプライト発注書 — 「わんこさんぽ」

## このゲームについて
どうぶつの森にインスパイアされた、ほのぼの系の2Dトップダウン（見下ろし型）ドット絵ゲーム。
女の子が柴犬を連れて小さな街を散歩し、動物の住民とおしゃべりしたり、木をゆすってりんごを
採ったり、池で釣りをしたりする。敵も失敗もない、やさしい雰囲気のゲーム。

現在はコード内蔵の簡易ドット絵で動いており、これをPNGスプライトシートに差し替えたい。
**下記のキャラクター10体すべて**のスプライトシートを image2 で生成してほしい。

## 共通アートルール（全キャラ厳守）
- 本物のピクセルアート：ドットが正確な格子に乗っていること。アンチエイリアス・グラデーション・ぼかし禁止
- ベタ塗りのみ、1キャラ最大8色
- 全体を囲む1pxの濃い茶色アウトライン（#4a3228）
- パステル調パレット基準色：クリーム #fff3e0 / ピンク #ff8fae / 緑 #8ecf6f / 水色 #aee3f5 / 黄 #ffd166
- 体型：2頭身。頭が全高の約60%。大きくて丸い黒目、ピンクのチーク（ほっぺ）
- 動物の住民は全員セーターなどの服を着ている
- 雰囲気：子ども向け絵本のようにあたたかく、かわいく
- **任天堂の既存キャラ（しずえ・たぬきち等）のコピーは禁止。あくまでオリジナルキャラ**
- 背景は完全透過のPNG

## シートレイアウト（全キャラ共通・厳守）
- 1キャラ = 1ファイル
- **3列 × 3行のグリッド**。全セルが同じサイズで等間隔、キャラの足元は同じベースラインに揃える
  - 行1：正面向き —（立ちポーズ / 歩き・右足前 / 歩き・左足前）
  - 行2：背面向き —（立ちポーズ / 歩き・右足前 / 歩き・左足前）
  - 行3：**左向き**の横 —（立ちポーズ / 歩きA / 歩きB）
- ※右向きはゲーム側で左右反転するので不要
- 歩き2コマは**必ず逆の足が前**になっていること（同じポーズ2枚はNG）
- キャラの身長は全キャラで統一（セル高の約80%）。犬のみ横長の体型

## 制作手順
1. **最初に「ミミィ（うさぎ）」を1枚作る** → これを全体のスタイル基準にする
2. 2体目以降は**必ずミミィのシートを参照画像として添付**し、
   「in the exact same art style, same proportions, same cell layout as this reference sheet」を指定する
3. 出来上がりを確認し、グリッドが崩れていたら・歩き2コマが同じポーズだったら再生成する
4. ファイル名は `sprites/` フォルダに以下の通り保存：
   girl.png / dog.png / mimii.png / moka.png / mike.png / kerota.png /
   penta.png / tonko.png / meeko.png / noron.png

## 納品前チェックリスト（各キャラ）
- [ ] 背景が完全に透過している（白背景や市松模様が焼き込まれていない）
- [ ] 3×3の全セルが同じサイズで、足元が揃っている
- [ ] 歩き2コマの前に出ている足が左右逆になっている
- [ ] 輪郭がにじんでいない（拡大してドットの格子を確認）
- [ ] ミミィと頭身・タッチが揃っている
