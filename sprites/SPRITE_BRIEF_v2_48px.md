# ドット絵スプライト発注書 v2（48px・高精細版）— 「わんこさんぽ」

## 前回からの変更点（まずこれを読む）
前回（32pxセル・96×96シート）の10体は合格済み。今回は**同じキャラたちの高精細版**を作る。
- セルサイズ：32px → **48px**（シート全体は **144×144px**、3列×3行）
- キャラの身長：**約40px**（セル高の約85%）。**足元は全コマ y=44 のベースラインに揃える**
- ドット数が増えるぶん、**表現を追加してよい**：
  - セーターの編み目・柄（ハート、ボーダー、水玉など）
  - 目のハイライト（白1〜2ドット）
  - 毛の2トーン表現（明部＋暗部。グラデーションは禁止、あくまでベタ塗り2色）
  - ほっぺのチーク、しっぽや耳の内側の色
- 使用色は1キャラ**最大12色**まで（前回は8色）
- **キャラのデザイン・服・色は前回と同じにする**（別人にしない）。前回の各シートを参照画像として必ず添付すること

## ⚠️ 前回の反省点（厳守）
前回、**横向きの行がキャラによって左向き/右向きバラバラ**だった（犬だけ左向き）。
今回は**全キャラ必ず「右向き」**にすること。1体できるごとに行3の向きを確認し、
左向きになっていたら再生成する。

## 共通アートルール（前回と同じ）
- 本物のピクセルアート：正確な格子、アンチエイリアス・グラデーション・ぼかし禁止
- 全体を囲む1pxの濃い茶色アウトライン（#4a3228）
- パステル調パレット基準：クリーム #fff3e0 / ピンク #ff8fae / 緑 #8ecf6f / 水色 #aee3f5 / 黄 #ffd166
- 2頭身、頭が全高の約60%、大きな丸い目
- 任天堂の既存キャラのコピー禁止。オリジナルキャラ
- 背景は完全透過PNG（市松模様を焼き込まない）

## シートレイアウト（厳守）
- 1キャラ = 1ファイル、**144×144px**、3列×3行（各セル48×48px）
  - 行1：正面 —（立ち / 歩き・右足前 / 歩き・左足前）
  - 行2：背面 —（立ち / 歩き・右足前 / 歩き・左足前）
  - 行3：**右向き**の横 —（立ち / 歩きA / 歩きB）※左向きはゲーム側で反転
- 歩き2コマは必ず逆の足が前（腕も逆に振れるとなお良い）
- 全コマ足元ベースライン y=44、キャラ中心はセル中央

## 制作手順
1. **ミミィ（うさぎ）から作る**。前回の `sprites/mimii.png` を参照画像に添付し、
   「same character, same outfit, higher resolution 48px version」と指定
2. 以降のキャラは「新しく作ったミミィv2」＋「前回のそのキャラのシート」の**2枚を参照画像に**
3. チェックリストで検品 → NG なら再生成
4. 保存先：`sprites/48/` フォルダに同じファイル名で保存
   （girl.png / dog.png / mimii.png / moka.png / mike.png / kerota.png /
    penta.png / tonko.png / meeko.png / noron.png）
